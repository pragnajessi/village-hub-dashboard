# VillageAPI Dashboard

A production-grade SaaS dashboard for India's complete village-level geographic data API platform.

## Features
- Admin Dashboard with analytics charts
- User Management with search and filters
- API Logs viewer with CSV export
- B2B User Portal with API key management
- Demo Contact Form with village autocomplete
- Dark/Light mode toggle
- Fully responsive design

## Tech Stack
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts for data visualization
- Vite build tool

## Getting Started
```bash
npm install
npm run dev
```
